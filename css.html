PROPIEDADES DE FORMATACAO DE TEXTO

As propriedades de formatação de texto CSS são usadas para formatar o texto e estilizar o texto.


A propriedade font-weight é utilizada para aplicar ou remover negrito de um elemento de texto. O valor de font-weight pode ser numérico ou uma palavra.

A propiedade font-size especifica o tamanho de uma fonte


A propriedade font-family do CSS permite que se faça uma lista de prioridades de familias de fontes e/ou nomes genéricos de famílias a serem especificados para um elemento selecionado. Ao contrário da maioria das demais propriedades CSS, os valores são separados por vírgula para indicar quais são as alternativas

PROPIEDADES DE COR

A propriedade color definine o valor da cor de um elemento com seu conteúdo em texto e seus decorações


A propriedade Background permite que você controle a cor do plano de fundo de um elemento, configure uma imagem como plano de fundo, e posicione uma imagem em uma página.
